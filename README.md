## 📦 Что это?

Этот проект — полноценный клон пакетного менеджера **Pacman** из Arch Linux, написанный на чистом **PowerShell** (без C#). Позволяет управлять программами из командной строки: устанавливать, удалять, искать, обновлять пакеты.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 📥 **Установка** | Установка пакетов с автоматической обработкой зависимостей |
| 🗑️ **Удаление** | Чистое удаление установленных пакетов |
| 🔍 **Поиск** | Поиск по имени, описанию и категории |
| 📋 **Список** | Просмотр всех установленных пакетов |
| ℹ️ **Информация** | Детальная информация о любом пакете |
| 🔄 **Обновление** | Обновление базы данных и всех пакетов |
| 💾 **Экспорт/Импорт** | Сохранение и восстановление списка пакетов |
| 🧹 **Очистка** | Удаление временных файлов кэша |
| 📊 **Статистика** | Графики и аналитика по пакетам |
| 📝 **Логирование** | Все действия записываются в лог-файл |
